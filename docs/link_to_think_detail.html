Link-To-Think

Description

At many different universities, there is a limited communication and collaboration opportunities between professors and students from differnt departments. Many professors' research projects would benefit from students' skills outside of their primary department, such as design or computer skills. Link to Think is a web platform that connects students with research opportunities posted by professors, fostering cross-disciplinary collaboration by enabling professors to discover and contact potential student researchers, streamlining academic research participation.

Implementation

The web is built with HTML, CSS, and Javascript for all front-end display and functionality. The content itself is pulled from JSON files.
